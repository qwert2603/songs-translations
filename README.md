# songs-translations

Приложение для просмотра переводов песен для Android.
Списки исполнителей и песен кешируются.
Есть возможность сохранения переводов в память телефона.
Есть возможность очиcтки кеша.
Есть кнопка "поделиться переводом".
Есть поиск по отдельному списку и по всем исполнителям/песням сразу.

Скриншоты в папке "screenshots".
